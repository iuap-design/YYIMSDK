6.公众号

6.1.获取我关注订阅号&广播号列表

/**
 * 获取广播号/订阅号列表[pubaccount]
 * @param arg {
 * success: function, //成功回调函数
 * error: function,  //失败回调函数
 * complete:function //无论成功失败都回调的函数
 * }
 */
YYIMChat.getPubAccount(arg);

6.2.查找订阅号&广播号

/**
 * 查找广播号/订阅号
 * @param arg {
 * keyword, 
 * success: function, 
 * error: function,
 * complete: function
 * }
 */
YYIMChat.queryPubaccount(arg);

6.3.关注订阅号

/**
 * 关注订阅号
